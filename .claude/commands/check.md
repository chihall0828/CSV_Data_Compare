# /check — 環境セットアップ＆ビルド・バリデーション確認

AGENTS.md の Setup / Validation 手順に従い、現在の環境が正常に動作するか確認します。

## 実行手順

以下をこの順番で実行してください。

1. `bash scripts/setup-codex.sh` を実行する（依存インストール → ソースツリー検証 → ビルド → バリデーション）
2. 終了コードが 0 であることを確認する
3. `git status --short` を実行して、余計な変更が生じていないことを確認する

## 報告フォーマット

実行後、以下の形式で結果を報告してください。

| ステップ | 結果 |
|---|---|
| npm install | 成功 / 失敗（エラー内容） |
| vite build | 成功 / 失敗（エラー内容） |
| validate-real-data.mjs | 成功 / 失敗（エラー内容） |
| validate-excel-and-calculation.mjs | 成功 / 失敗（エラー内容） |
| git status | クリーン / 変更あり（一覧） |

エラーが発生した場合は、エラーメッセージを引用し、原因と対処方法を提示してください。

## 注意

- `node_modules/` や `dist/` が生成されても、`.gitignore` で除外されているため git status に影響しない
- このコマンドはコードを変更しない。問題を発見したら報告のみ行い、修正は `/improve` または手動で行う
