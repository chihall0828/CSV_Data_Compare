# 系列別スタイルと出力設定

## 初めて使うときの順序

右側の「グラフ設定」を上から順に操作します。

1. **グラフの種類**: Time Series Plot / XY Plot と、Line / Scatter / Bar を選びます。
2. **表示するデータ**: Time Series PlotではX軸列とY軸列を選びます。XY Plotでは各ファイルカードのデータ設定からXY列を選びます。
3. **軸と見た目**: 線形・対数スケール、線幅、点表示、ラベル、表示範囲を設定します。
4. **画像保存・設定**: PNGの幅、高さ、解像度倍率、背景を指定して保存します。

普段使わない項目は折りたたまれているため、必要な設定だけ開いて操作できます。

## CSV内の系列ごとに色を変える

1. CSVまたはExcelを読み込みます。
2. 対象ファイルカードの **データ設定を開く** を押します。
3. Time Series Plotでは表示するY列、XY PlotではXY X・XY Yを選びます。
4. **系列ごとの色・点** で、各系列の **系列色** を指定します。
5. **自動** を押すと、CSV全体の初期スタイルまたは自動配色へ戻ります。

ファイルカード内の **CSV全体の初期スタイル** は初期値です。系列ごとの設定がある場合は、系列別設定が優先されます。

## 散布図の点を調整する

Scatterを選ぶか、MarkersをOnにすると、系列別に次の項目を変更できます。

- **点の色**: 線の色とは別に指定できます。
- **点サイズ**: 1〜30で指定できます。空欄の場合は「既定の点サイズ」を使用します。

## 軸スケール

「軸と見た目」でX軸・Y軸を **線形** または **対数** に切り替えられます。

- 日時や文字列のX軸では、X軸の対数設定は使用できません。
- 対数軸では0以下の値は表示されません。
- XY軸の等倍表示は、X軸とY軸がどちらも線形のときに使用できます。

## PNG画像サイズ

「画像保存・設定」で次を個別に指定できます。

- 幅: 480〜3000 px
- 高さ: 320〜2200 px
- 解像度倍率: 1× / 2× / 3×
- 背景: White / Transparent

実際の保存ピクセル数は、指定した幅・高さに解像度倍率を掛けた値です。たとえば1200 × 700 px、2×の場合は2400 × 1400 pxで保存されます。
